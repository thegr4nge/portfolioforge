    2. `get_connection(db_path: str = "data/market.db") -> sqlite3.Connection` function:
       - Opens with `check_same_thread=False`
       - Enables WAL mode: `conn.execute("PRAGMA journal_mode=WAL")`
       - Enables FK enforcement: `conn.execute("PRAGMA foreign_keys=ON")`
       - Calls `run_migrations(conn)` before returning

    3. `run_migrations(conn: sqlite3.Connection) -> None` function:
       - Reads current version: `conn.execute("PRAGMA user_version").fetchone()[0]`
       - For each unapplied migration (index `current` onwards): `conn.executescript(sql)` then `conn.execute(f"PRAGMA user_version = {i}")`
       - Each migration executes in a single `executescript` call (which is auto-committed)
       - Log with loguru: `logger.info("Applied migration {i} of {total}")`

    Type annotations everywhere. No magic numbers — use `CURRENT_SCHEMA_VERSION = len(MIGRATIONS)` constant.
