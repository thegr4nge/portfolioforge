    ```python
    from typing import Protocol, runtime_checkable

        async def fetch_ohlcv(
            self, ticker: str, from_date: date, to_date: date
        ) -> list[OHLCVRecord]: ...

        async def fetch_dividends(
            self, ticker: str, from_date: date, to_date: date
        ) -> list[DividendRecord]: ...

            self, ticker: str, from_date: date, to_date: date
        ) -> list[SplitRecord]: ...
    ```

    **`src/market_data/adapters/polygon.py`:**

    Class `PolygonAdapter`:

    ```python
    BASE_URL = "https://api.polygon.io"
    _MIN_INTERVAL_SECS: float = 12.0  # 5 req/min = 1/12s (conservative)
    ```

    `__init__(self, api_key: str) -> None`:
    - Store `self._api_key = api_key`
    - Create `self._semaphore = asyncio.Semaphore(1)` — inside __init__, NOT at module level (avoids RuntimeError in Python 3.12+)
    - Create `self._client = httpx.AsyncClient(timeout=30.0)`
    - Set `self.source_name = "polygon"`

    Private `async _get(self, url: str, params: dict[str, str]) -> dict`:
    - Acquires semaphore before request
    - `response = await self._client.get(url, params=params)`
    - `response.raise_for_status()` — let HTTPStatusError propagate
    - Parse JSON
    - `await asyncio.sleep(_MIN_INTERVAL_SECS)` INSIDE the semaphore context (not outside)
    - Return parsed dict

    Private `async _get_all_pages(self, url: str, params: dict[str, str]) -> list[dict]`:
    - Initialize `results: list[dict] = []`
    - `while url:` loop — call `_get(url, params)`, extend results from `data.get("results", [])`
    - Next iteration: `url = data.get("next_url", "")`, `params = {}` (params encoded in next_url)
    - Return results

    `async fetch_ohlcv(self, ticker: str, from_date: date, to_date: date) -> list[OHLCVRecord]`:
    - URL: `f"{BASE_URL}/v2/aggs/ticker/{ticker}/range/1/day/{from_date.isoformat()}/{to_date.isoformat()}"`
    - Params: `{"adjusted": "false", "sort": "asc", "limit": "50000", "apiKey": self._api_key}`
    - For each result dict, convert:
      - `t` (Unix ms) → date string: `date.fromtimestamp(r["t"] / 1000, tz=timezone.utc).date().isoformat()`
      - Map o→open, h→high, l→low, c→close, v→volume (cast to int)
      - `adj_close = r["c"]` (same as close — unadjusted; adj_factor applied by AdjustmentCalculator)
      - `adj_factor = 1.0` (default; updated when splits are detected)
      - `security_id = 0` — placeholder; Orchestrator replaces with real ID before writing
    - Return `list[OHLCVRecord]`

    `async fetch_dividends(self, ticker: str, from_date: date, to_date: date) -> list[DividendRecord]`:
    - URL: `f"{BASE_URL}/v3/reference/dividends"`
    - Params: `{"ticker": ticker, "limit": "1000", "sort": "ex_dividend_date", "apiKey": self._api_key}`
    - Map Polygon fields: `cash_amount→amount`, `ex_dividend_date→ex_date`, `pay_date→pay_date`, `record_date→record_date`, `declaration_date→declared_date`, `currency→currency`, `distribution_type→dividend_type`
    - `franking_credit_pct = None` for all records (Polygon does not provide this)
    - Filter results where `ex_dividend_date` is within [from_date, to_date] range
    - `security_id = 0` placeholder

    `async fetch_splits(self, ticker: str, from_date: date, to_date: date) -> list[SplitRecord]`:
    - URL: `f"{BASE_URL}/v3/reference/splits"`
    - Params: `{"ticker": ticker, "limit": "1000", "apiKey": self._api_key}`
    - Map: `execution_date→ex_date`, `split_from→split_from`, `split_to→split_to`
    - Filter results where `execution_date` within [from_date, to_date]
    - `security_id = 0` placeholder

    `async def close(self) -> None`: `await self._client.aclose()`

    All methods: full type annotations, loguru debug logging for each page fetched.
