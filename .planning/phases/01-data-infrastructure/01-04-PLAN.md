<objective>
Implement YFinanceAdapter for ASX securities (OHLCV + dividends) and AUD/USD FX rates — both using yfinance as the data source for Phase 1 prototype.

Purpose: DATA-02, DATA-03, and DATA-05 require ASX and FX data. yfinance is the approved prototype source (not production). The adapter wraps sync yfinance calls in async wrappers so the rest of the pipeline treats both adapters uniformly. Key pitfall: yfinance DataFrame indices are timezone-aware and must be normalized to UTC dates before storing.

Output: YFinanceAdapter in yfinance.py, FX rate fetching via AUDUSD=X, and a test suite that mocks yfinance.Ticker to avoid real network calls in CI.
</objective>

    Class `YFinanceAdapter`:

    `__init__(self) -> None`:
    - `self.source_name = "yfinance"`
    - `self._sleep_secs: float = 1.0`  — conservative guard between calls (no official rate limit)

    Private `_yf_ticker(self, symbol: str) -> yf.Ticker`:
    - Returns `yf.Ticker(symbol)` — factored out to enable monkeypatching in tests

    **`async fetch_ohlcv(self, ticker: str, from_date: date, to_date: date) -> list[OHLCVRecord]`**:
    - ASX suffix: `yf_symbol = ticker if ticker.endswith(".AX") else f"{ticker}.AX"`
    - `t = self._yf_ticker(yf_symbol)`
    - `df = t.history(start=from_date.isoformat(), end=to_date.isoformat(), interval="1d", auto_adjust=False, actions=False)`
    - If df is empty, log warning and return `[]`
    - Timezone normalization: `df.index.tz_convert("UTC").date` — gives a list of `datetime.date` objects
    - For each row:
      - `date_str = ts.tz_convert("UTC").date().isoformat()`
      - Open/High/Low/Close from DataFrame columns (cast to float)
      - Volume cast to int
      - `adj_close = float(row["Close"])` — raw close; adj_factor applied later by AdjustmentCalculator
      - `adj_factor = 1.0`
      - `security_id = 0` placeholder
    - `await asyncio.sleep(self._sleep_secs)` after the call
    - Return `list[OHLCVRecord]`

    **`async fetch_dividends(self, ticker: str, from_date: date, to_date: date) -> list[DividendRecord]`**:
    - `yf_symbol = ticker if ticker.endswith(".AX") else f"{ticker}.AX"`
    - `t = self._yf_ticker(yf_symbol)`
    - `series = t.dividends` — returns a Series with DatetimeTZDtype index and float values
    - If series is empty, return `[]`
    - For each (ts, amount) in series.items():
      - Normalize date: `ex_date = ts.tz_convert("UTC").date().isoformat()`
      - Filter: skip if ex_date < from_date.isoformat() or ex_date > to_date.isoformat()
      - Create DividendRecord with: security_id=0, ex_date=ex_date, amount=float(amount), currency="AUD", dividend_type="CD"
      - CRITICAL: `franking_credit_pct = None` — yfinance does NOT provide franking data for ASX stocks. Do not attempt to populate this field. Add a code comment: "# yfinance does not expose ASX franking credits; Phase 1 prototype stores None"
    - `await asyncio.sleep(self._sleep_secs)`
    - Return list

    **`async fetch_splits(self, ticker: str, from_date: date, to_date: date) -> list[SplitRecord]`**:
    - Same pattern as dividends but use `t.splits` Series
    - yfinance splits Series contains the split ratio (new/old) as a float — note this in a comment
    - Create SplitRecord with split_to=ratio, split_from=1 (yfinance encodes the ratio as a single number)
    - Return list (may be empty for many ASX stocks)

    **`async fetch_fx_rates(self, from_ccy: str, to_ccy: str, from_date: date, to_date: date) -> list[FXRateRecord]`**:
    - This is an additional method beyond the Protocol (FX isn't in DataAdapter Protocol since not all adapters support it)
    - Symbol: `f"{from_ccy}{to_ccy}=X"` (e.g., "AUDUSD=X")
    - `t = self._yf_ticker(f"{from_ccy}{to_ccy}=X")`
    - `df = t.history(start=from_date.isoformat(), end=to_date.isoformat(), interval="1d", auto_adjust=False, actions=False)`
    - For each row, use Close price as the exchange rate
    - Normalize date to UTC as with OHLCV
    - Create FXRateRecord(date=date_str, from_ccy=from_ccy, to_ccy=to_ccy, rate=float(row["Close"]))
    - `await asyncio.sleep(self._sleep_secs)`
    - Return list

    All methods: loguru debug logging. Full type annotations.
