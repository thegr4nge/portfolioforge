    **`@dataclass class IngestionResult`**:
    - `ticker: str`
    - `ohlcv_records: int`
    - `dividend_records: int`
    - `split_records: int`
    - `splits_detected: int`
    - `gaps_fetched: int`
    - `errors: list[str]`

    **`class IngestionOrchestrator`**:

    `__init__(self, conn: sqlite3.Connection) -> None`:
    - `self._conn = conn`
    - `self._writer = DatabaseWriter(conn)`
    - `self._tracker = CoverageTracker(conn)`
    - `self._adjuster = AdjustmentCalculator(conn)`

    **`async def ingest_ticker(self, ticker: str, adapter: DataAdapter, from_date: date, to_date: date) -> IngestionResult`**:

    1. Ensure security exists: `_get_or_create_security(ticker, adapter.source_name)`:
       - Check if ticker in securities table
       - If not: insert minimal record with `ticker=ticker, exchange="UNKNOWN", currency="USD"` (exchange/currency resolved later or from adapter metadata)
       - Return security_id

    2. For data_type in `["ohlcv", "dividends", "splits"]`:
       - Call `self._tracker.get_gaps(security_id, data_type, adapter.source_name, from_date, to_date)`
       - For each gap DateRange:
         - Try:
           - Fetch from adapter (call appropriate method based on data_type)
           - Assign `security_id` to all returned records (replace placeholder 0)
           - Write records via writer
           - Record coverage: `self._tracker.record_coverage(security_id, data_type, adapter.source_name, gap.from_date, gap.to_date, len(records))`
           - Log success to ingestion_log
         - Except Exception as e:
           - Log error to ingestion_log (status="error", error_message=str(e))
           - Append to result.errors
           - Continue (do not abort remaining gaps)

    3. After splits are written: for each new SplitRecord, call `self._adjuster.recalculate_for_split(security_id, split)`. Log count.

    4. Return IngestionResult with totals.

    **`def _get_or_create_security(self, ticker: str, source: str) -> int`**:
    - SELECT id FROM securities WHERE ticker=?
    - If not found: INSERT and return lastrowid
    - Log at debug level

    **`def _log_fetch(self, ticker: str, data_type: str, from_date: date, to_date: date, records: int, status: str, error: str | None = None) -> None`**:
    - Calls `self._writer.write_ingestion_log(IngestionLogRecord(...))`

    All methods: type annotations, loguru logging. No pandas.
