    ```python
    from typing import Optional
    import typer
    from rich.console import Console
    from rich.table import Table
    from market_data.db.schema import get_connection
    from market_data.quality.flags import QualityFlag

    status_app = typer.Typer(help="Inspect data coverage and quality")
    console = Console()
    ```

    **`@status_app.callback(invoke_without_command=True) def status_default(ctx, ticker: Optional[str] = Argument(None), db_path: str = Option("data/market.db"))`**:

    If no ticker provided (`market-data status`):
    - Query: join `securities` + `ingestion_coverage` + `ingestion_log` for summary per ticker
    - Display Rich Table: columns = [Ticker, Exchange, Coverage From, Coverage To, Total Records, Last Fetched, Quality Flags]
    - "Quality Flags" column: count of rows with non-zero quality_flags from ohlcv table
    - If no tickers in DB: print "No data ingested yet"

    If ticker provided (`market-data status AAPL`):
    - Show per-ticker detail:
      - Coverage by data_type (OHLCV, dividends, splits) with date ranges
      - Total records per data_type
      - Quality flag breakdown: how many rows have each flag (ZERO_VOLUME: N, PRICE_SPIKE: M, etc.)
      - Last 3 ingestion_log entries for this ticker

    **`@status_app.command("quality") def quality_command(ticker: str, db_path: str = Option("data/market.db"))`**:
    (`market-data quality AAPL`)
    - Query: `SELECT date, open, high, low, close, volume, quality_flags FROM ohlcv JOIN securities ON ... WHERE ticker=? AND quality_flags != 0 ORDER BY date`
    - Rich Table: Date | Open | High | Low | Close | Volume | Flags (decoded as flag names, comma-separated)
    - Decode flags: for each QualityFlag member, check if bit set, collect names
    - If no flagged rows: print "No quality issues found for AAPL"

    **`@status_app.command("gaps") def gaps_command(ticker: str, db_path: str = Option("data/market.db"))`**:
    (`market-data gaps AAPL`)
    - Use CoverageTracker to find gaps between recorded coverage intervals and today
    - Show gaps per data_type (OHLCV, dividends, splits) in a Rich Table: Data Type | Gap From | Gap To | Days Missing
    - If no gaps: print "Coverage complete for AAPL"

    All commands: handle `FileNotFoundError` if DB doesn't exist (print friendly message, exit 1).
