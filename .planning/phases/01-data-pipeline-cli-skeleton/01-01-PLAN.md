  <action>
    Create the `src/portfolioforge/` package with all subpackage directories. Each subpackage gets an `__init__.py` (data, engines, services, output can be empty for now).

    **config.py** — Application constants and defaults:
    - `DATA_DIR = Path.home() / ".portfolioforge"` (where SQLite cache lives)
    - `CACHE_DB_PATH = DATA_DIR / "cache.db"`
    - `CACHE_TTL_HOURS = 24` (market data TTL)
    - `FX_CACHE_TTL_HOURS = 1` (FX rates refresh more often)
    - `DEFAULT_RISK_FREE_RATE = 0.04` (approximate RBA cash rate)
    - `DEFAULT_BENCHMARKS = {"S&P 500": "^GSPC", "ASX 200": "^AXJO", "MSCI World": "URTH"}` (dict with display name -> ticker)
    - `DEFAULT_PERIOD_YEARS = 10` (default lookback for historical data)
    - `MAX_TICKERS_PER_FETCH = 20` (yfinance rate limit mitigation)
    - `SUPPORTED_MARKETS` dict mapping market suffix to currency: `{".AX": "AUD", "": "USD", ".L": "GBP", ".PA": "EUR", ".DE": "EUR"}`
    - `FRANKFURTER_BASE_URL = "https://api.frankfurter.dev"`

    **models/types.py** — Core type definitions:
    - `Market` enum: ASX, NYSE, NASDAQ, LSE, EURONEXT (with suffix and currency properties)
    - `Currency` enum: AUD, USD, EUR, GBP
    - `TickerInfo` pydantic BaseModel: `symbol: str`, `name: str | None`, `market: Market`, `currency: Currency`, `is_benchmark: bool = False`
    - Helper function `detect_market(ticker: str) -> Market` that infers market from ticker suffix (.AX = ASX, .L = LSE, etc., default NYSE/NASDAQ)
    - Helper function `detect_currency(ticker: str) -> Currency` using detect_market

    **models/portfolio.py** — Domain models:
    - `PriceData` pydantic BaseModel: `ticker: str`, `dates: list[date]`, `close_prices: list[float]`, `adjusted_close: list[float]`, `currency: Currency`, `aud_close: list[float] | None = None` (AUD-converted prices, populated after FX conversion)
    - `Holding` pydantic BaseModel: `ticker: str`, `weight: float` (0.0 to 1.0), `currency: Currency`
    - `Portfolio` pydantic BaseModel: `name: str`, `holdings: list[Holding]` with a `tickers` property returning list of ticker strings and a `weights_array` property returning numpy array. Add a validator that weights sum to approximately 1.0 (tolerance 0.01).
    - `FetchResult` pydantic BaseModel: `ticker: str`, `price_data: PriceData | None`, `error: str | None`, `from_cache: bool = False`

    **models/__init__.py** — Re-export all public models: `from .types import *` and `from .portfolio import *`

    **__init__.py** — Just `__version__ = "0.1.0"`

    Use pydantic BaseModel (not dataclasses) for all models since pydantic is already a dependency via typer. Use `model_config = ConfigDict(arbitrary_types_allowed=True)` where numpy arrays are involved.
  </action>
  <verify>

  </verify>
