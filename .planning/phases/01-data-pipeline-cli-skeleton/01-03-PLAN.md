    **data/currency.py** — Frankfurter API client and AUD conversion:

    - `fetch_fx_rates(base: str, quote: str, start: date, end: date, cache: PriceCache | None = None) -> pd.DataFrame`:
      1. Check cache first (via `cache.get_fx_rates`).
      2. On cache miss, call Frankfurter API: `GET {FRANKFURTER_BASE_URL}/{start}..{end}?from={base}&to={quote}`. Use `httpx.Client` (sync) with a 10-second timeout.
      3. Parse JSON response: `{"rates": {"2024-01-02": {"USD": 0.68}, ...}}`. Convert to DataFrame with DatetimeIndex and rate column.
      4. Store in cache.
      5. Return DataFrame.
      - Handle API errors: httpx.TimeoutException, httpx.HTTPStatusError, JSON decode errors. Return empty DataFrame on failure and log a warning.
      - Note: Frankfurter uses ECB data which only has business days and no weekend rates. This is fine since market data also only has business days.

    - `convert_prices_to_aud(price_data: PriceData, fx_rates: pd.DataFrame) -> PriceData`:
      1. If price_data.currency is AUD, return as-is (no conversion needed).
      2. Align fx_rates dates to price_data dates. For missing FX dates (holidays that differ between markets), forward-fill the last known rate.
      3. Multiply close_prices by the FX rate for each date. The Frankfurter rate from AUD to USD gives "1 AUD = X USD", so to convert USD prices to AUD: `aud_price = usd_price / rate`. Be careful with direction:
         - Fetch rate for AUD->USD (e.g., 0.65 means 1 AUD = 0.65 USD)
         - To convert USD to AUD: `aud_amount = usd_amount / 0.65`
         - To convert EUR to AUD: same pattern with AUD->EUR rate
      4. Store converted prices in `price_data.aud_close`.
      5. Return the updated PriceData.

    - `get_required_fx_pairs(tickers: list[str]) -> list[tuple[str, str]]`:
      - Given a list of tickers, determine which FX pairs are needed.
      - Use `detect_currency()` from models.types.
      - AUD tickers need no FX. USD tickers need AUD/USD. EUR tickers need AUD/EUR. GBP tickers need AUD/GBP.
      - Return deduplicated list of (base, quote) tuples where base is always "AUD".

    **Update data/fetcher.py:**
    - Add a `fetch_with_fx(ticker: str, period_years: int = 10, cache: PriceCache | None = None) -> FetchResult` function that:
      1. Calls existing `fetch_ticker_data()` to get raw prices.
      2. If the ticker is not AUD-denominated, fetches FX rates and applies conversion via `convert_prices_to_aud()`.
      3. Returns the FetchResult with AUD-converted prices populated in `price_data.aud_close`.
    - Update `fetch_multiple()` to use `fetch_with_fx()` instead of `fetch_ticker_data()`, and batch FX fetches (fetch each needed FX pair once, then apply to all relevant tickers).

    **Update data/__init__.py** — add exports: `from .currency import fetch_fx_rates, convert_prices_to_aud`

    - Test `fetch_fx_rates` returns empty DataFrame on API timeout
    - Test `convert_prices_to_aud` for USD ticker: prices divided by AUD/USD rate

    - Test `convert_prices_to_aud` handles missing FX dates (forward-fill)

    - Test FX direction is correct: if AUD/USD rate is 0.65 and stock price is $100 USD, AUD price should be ~$153.85 AUD
