**Function signatures (all required):**

```python
def run_backtest(
    portfolio: dict[str, float],
    start: date,
    end: date,
    rebalance: str,
    initial_capital: float = 10_000.0,
    benchmark: str = "STW.AX",
    db_path: str = "data/market.db",
    risk_free_rate: float = 0.0,
) -> BacktestResult: ...

def _load_prices(
    conn: sqlite3.Connection,
    tickers: list[str],
    start: date,
    end: date,
) -> pd.DataFrame: ...

def _generate_rebalance_dates(
    start: date,
    end: date,
    rebalance: str,
    available_dates: set[date],
) -> set[date]: ...

def _execute_trade(
    date: date,
    ticker: str,
    delta_shares: int,
    price: float,
    brokerage: BrokerageModel,
    holdings: dict[str, float],
    cash: float,
) -> tuple[Trade, dict[str, float], float]: ...
```

**Implementation rules:**

1. `run_backtest` calls `validate_portfolio(portfolio)` before opening the DB connection. Any weight error surfaces immediately.

2. `_load_prices` SQL must include `WHERE quality_flags = 0` (from RESEARCH.md anti-patterns). Returns a DataFrame indexed by `date` objects (not Timestamps), one column per ticker. Raises `ValueError` if any ticker has zero qualifying rows in the date range. Also validates single-currency: read the `currency` column from the `securities` table for each ticker; if more than one distinct currency appears, raise `ValueError("Mixed-currency portfolios are not supported in Phase 2. All tickers must share the same currency.")`.

3. `_generate_rebalance_dates` uses `REBALANCE_FREQS = {"monthly": "ME", "quarterly": "QE", "annually": "YE", "never": None}`. For `"never"`, return `{start}` only. For other frequencies, use `pd.date_range(start=start, end=end, freq=freq)`. For each generated date, snap to the nearest prior date in `available_dates` using `max(d for d in available_dates if d <= rebalance_date)`. Always include `start` (snapped to the first available date on or after start). Raises `ValueError` if `rebalance` is not in the allowed set.

4. `_execute_trade` is the ONLY function that creates Trade objects and modifies holdings/cash. It must call `brokerage.cost(trade_value)`. BUY: `cash -= trade_value + brok_cost`. SELL: `cash += trade_value - brok_cost`. No optional cost parameter — brokerage is always applied. Use `math.floor()` for share quantity calculations (called by `_execute_rebalance` before this function).

5. Rebalance logic (`_execute_rebalance`, can be a private function or inlined): compute `total_value = cash + sum(holdings.get(t, 0) * prices[t] for t in portfolio)`. For each ticker: `target_shares = math.floor(total_value * weight / price)`. `delta = target_shares - current_shares`. Skip if delta == 0. Call `_execute_trade` for non-zero deltas.

6. The simulation loop iterates over `prices.index` (each date in chronological order). On each iteration, look up whether `current_date` is in `rebalance_dates`. Update `equity_curve[current_date] = cash + sum(holdings.get(t, 0) * prices.loc[current_date, t] for t in portfolio)`. **Never access** `prices.loc[future_date]` in the loop — the loop structure enforces this by construction.

7. Benchmark: run the same simulation loop with `{"benchmark_ticker": 1.0}` as the portfolio, same initial_capital and brokerage. Reuse `_load_prices`, `_generate_rebalance_dates`, `_execute_rebalance`. The benchmark goes through the SAME code path — not a shortcut calculation.

8. `_build_result` assembles `BacktestResult` using `PerformanceMetrics(total_return=..., cagr=..., max_drawdown=..., sharpe_ratio=...)` from metrics.py functions. `coverage` is a list of `DataCoverage` — one per ticker (portfolio tickers + benchmark), with `from_date`, `to_date`, and `records` computed from the actual DataFrame rows used.

9. Use `loguru.logger` for info-level log on run start/end (no print statements).

10. Update `src/market_data/backtest/__init__.py` to export the real `run_backtest` from engine.py (replace the stub).

Keep engine.py under 200 lines. If helper functions push it over, extract `_rebalance_helpers.py` — but only if necessary.
