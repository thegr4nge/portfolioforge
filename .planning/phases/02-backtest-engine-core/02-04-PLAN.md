Create `tests/test_backtest_lookahead.py`. The goal is tests that would FAIL if someone removed the temporal slice guard from the engine loop. Use the same in-memory DB fixture pattern as test_backtest_engine.py.

**Key insight from RESEARCH.md:** Look-ahead bias in a fixed-weight rebalancing engine manifests as: the equity curve value on day D being influenced by prices on day D+1 or later. The test exploits this by constructing a price series where day 2 has a dramatically different price from day 1.

**Test 1: `test_day1_equity_unaffected_by_day2_price`**

Construct a synthetic price series with TWO days:
- Day 1 (2024-01-02): VAS.AX price = 100.0
- Day 2 (2024-01-03): VAS.AX price = 1000.0 (10x price — impossible to miss if look-ahead exists)

Run `run_backtest({"VAS.AX": 1.0}, start=date(2024,1,2), end=date(2024,1,3), rebalance="never", initial_capital=10_000.0)`.

Assert:
- `result.equity_curve.iloc[0]` reflects Day 1 prices only (approximately: `10_000.0 - brokerage_cost + (shares * 100.0)`)
- `result.equity_curve.iloc[0]` is LESS THAN `10_000.0` (brokerage was paid) and MUCH LESS than if day-2 price (1000.0) were used
- Specifically: if look-ahead were present, the day-1 equity would be ~100x inflated. Assert `result.equity_curve.iloc[0] < 11_000.0` — this would be false if day-2 prices bled into day-1 valuation.
- `result.equity_curve.iloc[1]` should be ~10x the day-1 value (price jumped 10x on day 2).

Document the assertion clearly: `# If this assertion fails, the engine has look-ahead bias — day-2 price is bleeding into day-1 equity.`

**Test 2: `test_single_day_backtest_uses_only_open_price`**

Run `run_backtest({"VAS.AX": 1.0}, start=date(2024,1,2), end=date(2024,1,2), rebalance="never", initial_capital=10_000.0)` — a 1-day backtest.

Assert:
- `len(result.equity_curve) == 1`
- `result.equity_curve.iloc[0]` is less than `initial_capital` (brokerage paid on initial purchase)
- `len(result.trades) == 1` (one initial buy)
- `result.trades[0].cost > 0.0`

This test documents: a 1-day backtest cannot incorporate any future price data — there is no day 2 to look ahead to.
