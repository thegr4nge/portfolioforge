**Add to `src/portfolioforge/output/backtest.py`:**

Add function `render_cumulative_chart(result: BacktestResult) -> None`:

1. `plt.clear_figure()` -- CRITICAL: plotext uses global state, must clear first.
2. `plt.date_form("Y-m-d")` to set date parsing format.
3. Convert `result.dates` to ISO format strings: `[d.isoformat() for d in result.dates]`.
4. Plot portfolio line: `plt.plot(date_strings, result.portfolio_cumulative, label="Portfolio", color="green")`.
5. Plot each benchmark: iterate `result.benchmark_cumulative.items()`, use colors `["blue", "red", "cyan", "magenta"]` cycling with modulo.
6. Set title: `plt.title("Cumulative Returns (Growth of $1)")`.
7. Set labels: `plt.xlabel("Date")`, `plt.ylabel("Value ($)")`.
8. For large datasets (>1000 points), downsample to ~500 points for chart display using `series[::step]` where `step = len(series) // 500`. This keeps the chart responsive without losing the visual shape. Apply same downsampling to dates and all value series.
9. `plt.show()`.
