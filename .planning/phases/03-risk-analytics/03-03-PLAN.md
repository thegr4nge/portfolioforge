<task type="auto">
  <name>Task 1: Sector caching and data fetching</name>
  <files>
    src/portfolioforge/data/cache.py
    src/portfolioforge/data/sector.py
    src/portfolioforge/config.py
  </files>
  <action>
1. Add a `SECTOR_CACHE_TTL_DAYS: int = 90` constant to `config.py`. Sectors rarely change, so 90-day cache TTL is appropriate.

2. Extend `PriceCache` in `data/cache.py`:
   - Add a `sector_cache` table to `_init_db()`:
     ```sql
     CREATE TABLE IF NOT EXISTS sector_cache (
         ticker TEXT PRIMARY KEY,
         sector TEXT NOT NULL,
         quote_type TEXT NOT NULL,
         fetched_at TEXT NOT NULL
     );
     ```
   - Add `get_sector(self, ticker: str) -> tuple[str, str] | None`:
     - Query sector_cache for ticker where fetched_at is within SECTOR_CACHE_TTL_DAYS.
     - Return `(sector, quote_type)` tuple or None if not cached/expired.
   - Add `store_sector(self, ticker: str, sector: str, quote_type: str) -> None`:
     - INSERT OR REPLACE into sector_cache with current timestamp.

3. Create `src/portfolioforge/data/sector.py`:
   - `fetch_sectors(tickers: list[str], cache: PriceCache) -> dict[str, str]`:
     - For each ticker, check cache first via `cache.get_sector(ticker)`.
     - If cached: use the cached sector. For quote_type "ETF", use "ETF". For "INDEX", use "Index". Otherwise use the sector string.
     - If not cached: fetch via `yf.Ticker(ticker).info`.
       - Extract `quote_type = info.get("quoteType", "EQUITY")`.
       - If quote_type == "ETF", sector = "ETF". If quote_type == "INDEX", sector = "Index". Else `sector = info.get("sector", "Unknown")`.
       - Cache result via `cache.store_sector(ticker, sector, quote_type)`.
     - Wrap each yfinance call in try/except. On failure: sector = "Unknown", quote_type = "EQUITY". Still cache the result (avoids retrying failed lookups every time).
     - Print a progress message to stderr for each uncached ticker: `[dim]Fetching sector data for {ticker}...[/dim]` (using Rich Console on stderr, same pattern as services/backtest.py).
     - Return `dict[str, str]` mapping ticker -> sector string.
  </action>

  </verify>
