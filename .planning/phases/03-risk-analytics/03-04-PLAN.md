- test_finds_worst_drawdowns: Verify periods are sorted by depth (worst first). The deepest drawdown (to 0.85) should be first.
- test_drawdown_has_correct_fields: Each period dict has keys: peak_date, trough_date, recovery_date, depth, duration_days, recovery_days.
- test_unrecovered_drawdown: The final drawdown should have recovery_date=None and recovery_days=None.
- test_top_n_limits_results: Pass top_n=2, verify only 2 periods returned.

Import from portfolioforge.engines.risk. Use pd.Timestamp for DatetimeIndex construction. All date comparisons use .date() since the engine returns date objects.
