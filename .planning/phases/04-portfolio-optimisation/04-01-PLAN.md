<task type="auto">
  <name>Task 2: Create optimisation engine with Ledoit-Wolf shrinkage, frontier generation, and scoring</name>
  <files>src/portfolioforge/engines/optimise.py</files>
  <action>
  Create `src/portfolioforge/engines/optimise.py` following the pure-function engine pattern from `engines/risk.py`. All functions take pandas primitives in, return dicts/lists out. No I/O, no display imports.

  Import from PyPortfolioOpt:
  ```python
  from pypfopt.expected_returns import mean_historical_return
  from pypfopt.risk_models import CovarianceShrinkage
  from pypfopt.efficient_frontier import EfficientFrontier
  ```

  Implement these functions:

  1. **`compute_optimal_weights(prices, weight_bounds, risk_free_rate=0.04)`** -> dict
     - Takes `prices: pd.DataFrame` (columns=tickers, DatetimeIndex), `weight_bounds: tuple[float, float]`
     - Computes `mu = mean_historical_return(prices)` and `S = CovarianceShrinkage(prices).ledoit_wolf()` (OPT-03)
     - Creates `EfficientFrontier(mu, S, weight_bounds=weight_bounds)`
     - Calls `ef.max_sharpe(risk_free_rate=risk_free_rate)`
     - Returns dict with `"weights": dict(ef.clean_weights())`, `"expected_return"`, `"volatility"`, `"sharpe_ratio"` from `ef.portfolio_performance()`

  2. **`compute_efficient_frontier(prices, weight_bounds, n_points=50, risk_free_rate=0.04)`** -> list[dict]
     - Computes `mu` and `S` ONCE (anti-pattern from research: don't recompute)
     - Finds feasible return range: create fresh EF for min_volatility, get min_ret; create fresh EF for max_sharpe, get max_ret
     - Loops `np.linspace(min_ret, max_ret * 1.05, n_points)`, for each target:
       - Create FRESH `EfficientFrontier` (Pitfall 1: EF is single-use)
       - Call `ef.efficient_return(target)` wrapped in try/except (Pitfall 3: infeasible targets)
       - Append `{"return": ret, "volatility": vol, "sharpe": sharpe}` from `ef.portfolio_performance()`
     - Return list of frontier point dicts

  3. **`score_portfolio(prices, weights, weight_bounds, risk_free_rate=0.04)`** -> dict
     - Computes `mu` and `S` once
     - Calculates user portfolio metrics: `user_ret = w @ mu`, `user_vol = sqrt(w @ S @ w)`, `user_sharpe`
     - Creates fresh EF, calls `ef.efficient_risk(user_vol)` to find optimal portfolio at same risk level
     - Gets `opt_ret, opt_vol, opt_sharpe` and `opt_weights = dict(ef.clean_weights())`
     - Computes `efficiency_ratio = user_ret / opt_ret` (clamped to [0, 1] range)
     - Returns dict with all user metrics, optimal metrics, optimal_weights, and efficiency_ratio

  IMPORTANT NOTES from research:
  - Always pass PRICES (not returns) to `mean_historical_return()` and `CovarianceShrinkage()` (Pitfall 4)
  - Create a NEW EfficientFrontier instance for EVERY optimisation call (Pitfall 1)
  - Use `float()` to convert numpy scalars to Python floats for Pydantic compatibility
  </action>
