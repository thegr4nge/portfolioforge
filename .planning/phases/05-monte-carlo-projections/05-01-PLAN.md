Create `engines/montecarlo.py` as a pure computation module (no I/O, no display) following the pattern in engines/backtest.py and engines/risk.py. All functions take numpy/pandas primitives and return numpy arrays or floats.

**Functions to implement:**

1. **`RISK_PROFILES`** dict constant:
   ```python
   RISK_PROFILES: dict[str, dict[str, float]] = {
       "conservative": {"sigma_scale": 0.7},
       "moderate": {"sigma_scale": 1.0},
       "aggressive": {"sigma_scale": 1.3},
   }
   ```
   No mu haircut -- research recommended sigma scaling only.

2. **`estimate_parameters(prices: pd.DataFrame, weights: np.ndarray) -> tuple[float, float]`**:
   - Compute weighted portfolio cumulative returns using `compute_cumulative_returns` from engines/backtest (reuse, don't rewrite).
   - Calculate daily log returns: `np.log(cumulative / cumulative.shift(1)).dropna()`
   - `mu = float(daily_log_returns.mean() * 252)`
   - `sigma = float(daily_log_returns.std() * np.sqrt(252))`
   - Return `(mu, sigma)`
   - Use log returns (not arithmetic) to avoid upward bias per research pitfall 2.

3. **`simulate_gbm(initial_value, mu, sigma, years, n_paths, monthly_contribution=0.0, seed=None) -> np.ndarray`**:
   - Exactly as in the research code example. Monthly time steps (dt = 1/12).
   - Use `np.random.default_rng(seed)` (NOT legacy np.random).
   - Drift = `(mu - 0.5 * sigma**2) * dt` (Ito correction is critical).
   - Without contributions: fully vectorised via cumsum.
   - With contributions: iterative loop across time but vectorised across paths.
   - Returns array shape `(n_paths, years * 12)`.

4. **`extract_percentiles(paths: np.ndarray, percentiles: list[int] | None = None) -> dict[int, np.ndarray]`**:
   - Default percentiles: `[10, 25, 50, 75, 90]`
   - `np.percentile(paths, p, axis=0)` for each p
   - Returns dict mapping percentile int to 1D array of length n_steps.

5. **`goal_probability(paths: np.ndarray, target: float, target_month: int) -> float`**:
   - `target_month` is 1-based (month 12 = end of year 1).
   - `final_values = paths[:, target_month - 1]`
   - Return `float(np.mean(final_values >= target))`
