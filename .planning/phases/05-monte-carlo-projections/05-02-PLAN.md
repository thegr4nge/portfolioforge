**Service** (`services/montecarlo.py`):

Create `run_projection(config: ProjectionConfig) -> ProjectionResult` following the pattern in services/backtest.py and services/optimise.py:

1. Reuse `_fetch_all` from `services/backtest` to fetch ticker prices (import it directly -- same pattern as services/optimise.py).
2. Call `align_price_data` from engines/backtest to get price DataFrame.
3. Call `estimate_parameters(prices, weights_array)` from engines/montecarlo to get mu, sigma.
4. Apply risk tolerance scaling: `sigma_adjusted = sigma * RISK_PROFILES[config.risk_tolerance.value]["sigma_scale"]` (import RISK_PROFILES from engines/montecarlo).
5. Call `simulate_gbm(config.initial_capital, mu, sigma_adjusted, config.years, config.n_paths, config.monthly_contribution, config.seed)`.
6. Call `extract_percentiles(paths)` to get percentile bands.
7. Compute `final_values`: for each percentile, take the last element of the array.
8. If `config.target_amount` and `config.target_years` are set:
   - Call `goal_probability(paths, config.target_amount, config.target_years * 12)`.
   - Compute median at target year: `percentiles[50][config.target_years * 12 - 1]`.
   - Compute shortfall: `max(0, config.target_amount - median_at_target)`.
   - Build `GoalAnalysis` model.
9. Build portfolio_name from ticker:weight pairs (same format as backtest service).
10. Return `ProjectionResult` with all fields populated. Convert numpy arrays to lists for percentiles dict.

**Output** (`output/montecarlo.py`):

Create `render_projection_results(result: ProjectionResult, console: Console) -> None`:

1. Print a header panel: "Monte Carlo Projection: {portfolio_name}"
2. Print simulation parameters table:
   - Initial Capital, Time Horizon, Monthly Contribution, Risk Tolerance, Paths, Estimated Return (mu), Estimated Volatility (sigma)
   - Format mu/sigma as percentages (e.g., "8.2%")
   - Format capital/contribution as currency (e.g., "$100,000")

3. Print percentile outcome table (same pattern as research code example):
   - Columns: "Percentile", then "Year N" for key intervals
   - Key intervals: filter [5, 10, 15, 20, 25, 30] to those <= result.years
   - Rows: 10th (pessimistic), 25th, 50th (median), 75th, 90th (optimistic)
   - Values formatted as "$XXX,XXX"
   - Color the 50th percentile row green (bold)
   - Color 10th row red, 90th row green

4. Print final value summary:
   - "At year {years}:" with each percentile value
   - Color: red for 10th, yellow for 25th, green for 50th, yellow for 75th, green for 90th

5. If `result.goal` is not None, print goal analysis panel:
   - "Target: ${target:,.0f} in {years} years"
   - "Probability of success: {prob:.1%}" -- green if >= 0.7, yellow if >= 0.4, red otherwise
   - "Median portfolio at target: ${median:,.0f}"
   - If shortfall > 0: "Shortfall from target: ${shortfall:,.0f}" in red
