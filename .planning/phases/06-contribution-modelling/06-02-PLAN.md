**engines/contribution.py** -- Add two functions to the existing file:

1. `compute_dca_vs_lump(prices: pd.DataFrame, weights: np.ndarray, total_capital: float, dca_months: int) -> tuple[pd.Series, pd.Series]`:
   - Compute daily portfolio returns from aligned multi-ticker prices
   - Lump sum: invest total_capital on day 1, track cumulative growth
   - DCA: invest total_capital / dca_months each month start (use `resample("MS").first()` for month start dates)
   - Each DCA tranche grows independently from its investment date
   - Returns (lump_sum_values, dca_values) as pd.Series indexed by date
   - Assume uninvested DCA capital earns 0% (conservative)

2. `rolling_dca_vs_lump(prices: pd.DataFrame, weights: np.ndarray, total_capital: float, dca_months: int, holding_months: int) -> dict[str, float]`:
   - Compute portfolio prices as weighted sum of ticker prices
   - Resample to monthly start prices
   - For each possible start date window (total_window = dca_months + holding_months):
     - Lump sum: buy at first price, value at last price
     - DCA: buy monthly_amount each month for dca_months periods, value at last price
   - Return `{"windows_tested": int, "lump_win_pct": float}`

Both functions take pandas/numpy primitives (not Pydantic models) following the engine convention.
