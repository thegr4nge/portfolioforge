# market-data

Local market data infrastructure for backtesting and financial analysis.

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -e ".[dev]"
```

## Structure

- `src/market_data/` — core library
- `tests/` — test suite
- `SPEC.md` — layer-by-layer implementation spec
- `CLAUDE.md` — project conventions for Claude Code
