# Outreach Messages
*Copy-paste ready. Do not paraphrase — these are tested and calibrated.*

---

## How to use this

1. Find the prospect on LinkedIn (URLs in the lead tracker)
2. Send a connection request first if not connected — use the note: *"Hi [Name] — I built something that might be useful for SMSF accountants. Worth a look?"*
3. Once connected, send the personalised message below
4. If no reply after 5–7 days, send Follow-up 1
5. If still no reply after another 5–7 days, send Follow-up 2 and move on
6. Mark everything in the lead tracker
